Любая операция с NaN возвращает NaN.

Пробельные символы по краям строк при преобразовании к числу не учитываются

Методы и Свойства

  • parseInt() / parseFloat()

    Преобразуют строку в число символ за символом пока это возможно(целое/дробное)

  • Math.floor / Math.ceil / Math.round / num.toFixed(точность)

    Округление: вниз / вверх / до ближайшего / до точности

  • Другие ункции Math.

    Math.acos(x) / Math.asin(x) / Math.atan(x) / Math.atan2(y, x) / Math.sin(x) / Math.cos(x) / Math.tan(x) / Math.sqrt(x) / Math.log(x) / Math.pow(x, exp) / Math.abs(x) / Math.exp(x) / Math.max(a, b, c...) / Math.min(a, b, c...) / Math.min(a, b, c...)

  • Math.random()

    Возвращает псевдо-случайное число в интервале [0,1) – то есть между 0(включительно) и 1(не включая).
